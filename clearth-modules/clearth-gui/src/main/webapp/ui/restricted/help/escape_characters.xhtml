\t - a tabulation character
\n - a newline
\r - a carriage return
\' - a single quote character
"" - a double quote character
\\\\ - a backslash character

, - comma

The whole value in the column with comma should be enclosed in double quotes

Examples

Value in column is: value,with,commas

should be written like this: "value,with,commas"

The same is for expressions: @{format(time(0), 'yyyy-MM-dd')}

In CSV it should be: "@{format(time(0), 'yyyy-MM-dd')}"
