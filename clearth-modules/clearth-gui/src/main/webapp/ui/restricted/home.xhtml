Welcome to ClearTH
ClearTH provides various features: test automation, connections to system gateways, converters, etc.
Please select needed function from the menu on the left:
  • Automation - run automated tests and see history of the launches including detailed reports.
  • Connectivity - setup connections to system gateways.
    •
  • ClearTH logs - download ClearTH logs and setup the verbosity of the loggers.
  • Tools - miscellaneous helpers like message parser, message to script converter, etc.
  • User management - administration of ClearTH users.
    •
  • Recent pop-ups - list of pop-up messages that were shown recently in case you have missed them.
  • Roll up - button to toggle compact view of the menu.
  • Sign out - log out from the application and return to log-in page.
